# IHL Referral Landing Page

Source of truth for the IHL referrer landing page prototype, review workflow, and eventual publishing handoff.

Live preview:

https://richie261.github.io/ihl-referral-landing-page/

## Purpose

This page is designed to give GPs, psychiatrists, paediatricians, and other healthcare referrers a fast, simple way to understand how to refer to IHL.

The intended pathway order is:

1. HealthLink first.
2. Online referral form second.
3. Intake support third.

## Files

- `index.html` - interactive referrer landing page prototype.
- `online-referral-form-v1.html` - draft online referral form experience.
- `docs/decisions.md` - decisions and scope boundaries.
- `docs/launch-checklist.md` - pre-launch review checklist.

## Operating Rule

This repository is for page copy, prototype interaction, review notes, assets, and publishing handoff only. Do not add patient/client clinical information or referral submissions here.

GitHub is the working source of truth. Webflow is the final publishing layer once the page is approved.
