Live preview:

https://richie261.github.io/ihl-referral-landing-page/

## Purpose

Current active focus: the referrer landing page.

This page is designed to give GPs, psychiatrists, paediatricians, and other healthcare referrers a fast, simple way to refer to IHL.

The current direction is restrained and form-first: a simple, professional and warm referral gateway, not a sales-led funnel.

The page presents all referral pathways clearly so referrers can choose what works for them:

1. HealthLink eReferral.
2. Online referral form.
3. Intake email.
4. Phone or fax support.

## Operating Rule

This repository is for page copy, prototype interaction, review notes, assets, and publishing handoff only. Do not add patient/client clinical information or referral submissions here.

GitHub is the working source of truth. Webflow is the final publishing layer once the page is approved.

Broader IHL Platform strategy, Practice Brain, Tacklit/vendor decision-making, HubSpot CRM rules, and internal operating-model detail should be kept in the strategy/master hub, not expanded inside this public landing page repo.
