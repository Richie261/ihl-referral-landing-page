# Public Build Context

Date: 2026-05-07

This repo is the public-facing referrer landing page and form prototype. It sits inside a wider IHL operating-model project, but should only contain public-safe material.

## What This Repo Should Do

- Help GPs, psychiatrists, paediatricians and allied referrers choose the right referral pathway.
- Keep HealthLink EDI `inshealh` easy to find and copy.
- Provide the online referral form.
- Provide intake email, phone and fax details.
- Explain what happens after referral.
- Improve referral quality with a short "what to include" checklist.
- Prepare copy/layout for Webflow.

## What This Repo Should Not Do

- Store patient/client referral submissions.
- Store clinical records, referral letters, risk details or attachments.
- Become the full Practice Brain specification.
- Contain vendor due-diligence notes for Tacklit, Halaxy, Zanda or custom build decisions.
- Contain private HubSpot sync or CRM implementation detail beyond the public data-boundary principle.

## Wider Workstreams

The wider work continues in separate internal files:

- IHL platform strategy.

Feed those items back to the strategy chat rather than expanding this repo.
